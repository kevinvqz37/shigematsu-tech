# Shigematsu Tech — Company Website

Static website for Shigematsu Tech (重松テック).
Deployed on Vercel at shigematsu-tech.vercel.app

## Pages
- `/` — Homepage (index.html) — trilingual JP/EN/ES
- `/dispatchos` — DispatchOS product page
- `/kevin` — Founder profile
- `/demo` — Product demo (served from chikugo-dispatch.vercel.app/demo)

## Stack
Pure HTML/CSS/JS — no build step required.
Vercel deploys automatically on push to main.
